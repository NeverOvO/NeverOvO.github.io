Flutter 更新

命令行： flutter upgrade （2021年3月7日 可以直接更新到2.0.1）

Flutter 启用桌面端

更新完成后启用三个桌面端：
flutter config --enable-windows-desktop
flutter config --enable-macos-desktop
flutter config --enable-linux-desktop

然后执行：flutter devices 确认现在支持列表，在Mac下如图即支持了macOS端。

Flutter 老项目增加桌面支持

//未成功实施：网上的例子是下列语句，可以在原本不支持桌面端的项目中增加桌面端支持。
flutter create --platforms=windows,macos,linux .

Flutter 老项目对于第三方包空安全支持的快速查询语句

命令行cd到项目root目录下，执行：
dart pub outdated --mode=null-safety

1、dart fix --dry-run
2、dart fix --apply

3: flutter doctor等待更新完成
